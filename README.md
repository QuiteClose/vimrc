# NeoVim (and Vim) Config

## NeoVim Installation

```bash
mkdir ~/etc
git clone https://github.com/quiteclose/vimrc.git ~/etc/vimrc
ln -s ~/etc/vimrc/neovim/nvim ~/.config/nvim
mkdir ~/.nvimundo
```
